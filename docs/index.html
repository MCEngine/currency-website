MCEngineCurrencyApi Documentation

Introduction

The `MCEngineCurrencyApi` is a robust API for managing in-game currency. This documentation will explain its functionalities and usage.

Features

  • Initialize and manage database connections.
  • Support for MySQL and SQLite databases.
  • CRUD operations for player data and currency management.

Code Usage


// Example Usage in Java
MCEngineCurrencyApi api = new MCEngineCurrencyApi("mysql", new String[] {
    "host", "port", "database", "user", "password"
});

api.initDB(); // Initializes the database
api.initPlayerData(UUID.randomUUID()); // Initializes player data
api.addCoin(playerUuid, "gold", 100.0); // Adds 100 gold coins
api.minusCoin(playerUuid, "silver", 50.0); // Removes 50 silver coins
api.disConnect(); // Disconnects from the database

Explanation of Methods

initDB()
Establishes a connection to the database and creates required tables.
initPlayerData(UUID uuid)
Initializes currency data for a new player.
addCoin(UUID uuid, String coinType, double amt)
Adds the specified amount of a coin type to a player.
minusCoin(UUID uuid, String coinType, double amt)
Subtracts the specified amount of a coin type from a player.
disConnect()
Disconnects from the database.
